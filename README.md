# HTML Fundamentals 🚀

Welcome to the **HTML Fundamentals** repository! This project serves as a comprehensive reference guide for absolute beginners, demonstrating the essential structural and formatting tags of HTML5.

---

## 📋 Features & Tags Covered

This repository contains practical examples of the following fundamental HTML elements:

### 1. Document Structure & Metadata
*   `<!DOCTYPE html>`: Defines the document type as HTML5.
*   `<html lang="en">`: Sets the primary language of the webpage.
*   `<head>` & `<meta>`: Handles character encoding (`UTF-8`) and ensures responsive viewport scaling for mobile devices.
*   `<title>`: Sets the browser tab title to *"My First Pages"*.

### 2. Navigation & Hyperlinks (`<a>`)
*   **Relative Paths:** Links to internal project files like `Layout Techniques` and `Pro Level`.
*   **Absolute Paths:** External links redirecting to official websites like **Google** and the **VS Code Download page**.

### 3. Text Structure & Headings
*   **Headings:** Implementation of structural hierarchy using all six levels (`<h1>` through `<h6>`).
*   **Paragraphs:** Standard text blocks formatted using the `<p>` tag.
*   **Line Breaks & Rules:** Using `<br/>` for explicit line breaks and `<hr>` for thematic horizontal lines/sections dividers.

### 4. Media & Images (`<img>`)
*   Embedding external web images using absolute URLs.
*   Embedding local project assets (e.g., `Notebook.jpg`) with specific dimension tuning (`height="50"`) and proper fallback `alt` description text.

### 5. Typography & Inline Formatting
*   `<b>`: Renders text in **BOLD**.
*   `<i>`: Renders text in *Italic*.
*   `<u>`: Applies an <u>underline</u> to specific text or elements within paragraphs.
*   `<big>` & `<small>`: Demonstrates presentational text sizing adjustments.

### 6. Advanced Formatting
*   **Subscript (`<sub>`):** Used for chemical formulas like $H_2O$.
*   **Superscript (`<sup>`):** Used for mathematical representations such as $3^2 = 9$ and $A^n + B$.
*   **Preformatted Text (`<pre>`):** Preserves both spaces and line breaks exactly as they are typed in the source code (demonstrated using Lorem Ipsum placeholder text).
